# Hardware Components List

This file details all the physical components used to build the mini drone MVP.

| Component                  | Item                                             | Quantity     | Notes                                                       |
| :------------------------- | :----------------------------------------------- | :----------- | :---------------------------------------------------------- |
| **Frame**                  | 2.5-inch "Toothpick" or 3-inch Quadcopter Frame  | 1            | Lightweight carbon fiber frame.                             |
| **Flight Controller (FC)** | F4 or F7 All-in-One (AIO) Flight Controller      | 1            | An AIO board includes the FC and ESCs in one unit.          |
| **Motors**                 | 1103 Brushless Motors (8000KV)                   | 4            | KV rating chosen for 2S or 3S LiPo batteries.               |
| **Propellers**             | 2.5-inch or 3-inch propellers                    | 8 (4 spares) | Must match the frame size.                                  |
| **Radio Transmitter**      | Radiomaster TX12 or similar multi-protocol radio | 1            | The remote control used by the pilot.                       |
| **Radio Receiver**         | ExpressLRS (ELRS) or Crossfire Nano Receiver     | 1            | Small, lightweight receiver with good range.                |
| **Battery**                | 2S 450mAh or 3S 300mAh LiPo Battery              | 2            | Lithium Polymer battery. At least two for testing and demo. |
| **Battery Charger**        | LiPo Balance Charger                             | 1            | Essential for safely charging LiPo batteries.               |
| **Connectors**             | XT30 Battery Connector                           | 1            | Soldered to the flight controller's power leads.            |
| **Miscellaneous**          | Wires, solder, zip ties, battery strap           | -            | For assembly and cable management.                          |
